| Feature | URL | Access | Notes |
|---------|-----|--------|-------|

| Change Password | `/change_password/` | Navbar (logged in) | Must verify current password |

## Key Security Features

âœ… **CSRF Protection** - All forms protected against CSRF attacks

âœ… **Password Validation** - Minimum 8 characters required

âœ… **Token-Based** - Reset links are one-time use with token verification

âœ… **Time Expiry** - Reset links expire after 24 hours

âœ… **Email Verification** - Only registered email accounts can reset

âœ… **Current Password Check** - Change password requires verifying current password

## Troubleshooting

### Not receiving reset email?
- Check spam/junk folder
- Verify email is associated with account
- Try again - click "Try Again" button on confirmation page

### Reset link expired?
- Links are valid for 24 hours
- Request a new reset link on the forgot password page

### Forgot current password and email not working?
- Contact admin for account recovery
- Manual password reset may be needed

### "Current password is incorrect" when changing?
- Make sure you're typing the password you use to login
- Try login again to verify password works
